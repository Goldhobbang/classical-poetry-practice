# 고전시가 현대어 풀이 연습기

고전시가나 문학 작품을 **현대어로 직접 풀이**하며 연습할 수 있는 웹 서비스입니다.  
원하는 작품 전문을 넣어 연습 템플릿으로 활용하고, 저장/불러오기를 통해 반복 학습할 수 있습니다.

## 주요 기능

- **작품 등록 및 저장**
  - 원하는 문학 작품 전문을 입력해 로컬 스토리지에 저장할 수 있습니다.
  - 저장된 작품을 다시 불러와 즉시 연습을 이어갈 수 있습니다.

- **현대어 풀이 입력 연습**
  - 원문을 문장(줄) 단위로 보면서 현대어 풀이를 작성할 수 있습니다.
  - 흐름 중심 UI로 앞서 작성한 내용, 현재 문장, 다음 문장을 확인할 수 있습니다.

- **결과 텍스트 취합 및 복사**
  - 입력이 끝난 현대어 풀이 문장 묶음을 한 번에 모아 확인할 수 있습니다.
  - 복사 가능한 텍스트 형태로 제공되어, 이후 AI 채점이나 기타 용도로 활용하기 쉽습니다.

## 활용 예시

- AI 기반 문학 풀이 첨삭/채점 요청
- 수행평가 대비용 자기 점검
- 작품별 해석 노트 초안 작성
